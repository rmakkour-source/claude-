## 💻 Command Reference for Claude Code
To use this suite, open your terminal, navigate to this folder, and launch `claude`. Use the following commands to trigger the AI Agents:

### 📊 Audits & Strategy
* `/market audit` - Launches 5 parallel agents for a complete Funnel, SEO, CRO, and Competitor breakdown.
* `/market quick` - Outputs a 60-second, brutal assessment of current bottlenecks straight to the terminal.
* `/market landing` - Audits GoHighLevel funnels specifically for the 40-65 yr old demographic (UX, contrast, Trustpilot anchors).

### ⚙️ GoHighLevel (GHL) Automation
* `/market ghl` - Maps out exact, node-by-node GHL workflows for Setup Rescue and Pipeline Reactivation.
* `/market aibot` - Generates the Master Prompt and Q&A training data for "Joe" the GHL Conversational AI Bot.

### 💰 Sales & Acquisition
* `/market compliance` - Writes "Ban-Proof" Meta Ads using dog-whistle copy to keep ad accounts alive.
* `/market ads` - Generates standard FB/IG direct-response ads and visuals.
* `/market whatsapp` - Generates the master vault of WhatsApp sales scripts and Lifetime Plan upsells.
* `/market retention` - Writes psychological crisis-communication scripts to blame ISPs (Virgin/Sky) for buffering, saving €250 refund requests.
